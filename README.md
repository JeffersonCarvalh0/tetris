# Building Instructions
In order to build the project, open the terminal in the root directory and do as
follows:

``` shell
mkdir build && cd build
cmake ..
make
```

You can execute the game with `./main`.

# Controls
* A - Move to the left
* D - Move to the right
* S - Move down
* W - Hold tetrimino
* Q - Rotate counterclockwise
* E - Rotate clockwise
* Space - Hard drop
* Esc - Close the game
